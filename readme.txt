== Description ==

A WordPress Plugin containing Masonry-styled alternatives to Gutenberg's Core Blocks. 

== Installation ==


1. Upload the plugin files to the `/wp-content/plugins/masonry-blocks` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
